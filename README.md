# Agriculture & Rainfall Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes rainfall patterns and agricultural productivity across Indian states.

## Features
- Rainfall trend analysis
- Crop production analysis
- Yield trend analysis
- Correlation analysis
- Interactive slicers
- Rainfall map visualization

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling

## Key Insights
- Higher rainfall regions generally show increased rice production.
- Rainfall variability impacts agricultural productivity.
- Rice yield improved steadily over the years.
